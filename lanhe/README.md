## 🚀 使用方法

### 查看文档
```bash
# 查看产品介绍
cat docs/产品介绍.md

# 查看功能指南
cat docs/功能使用指南.md

# 查看安装指南
cat docs/安装部署指南.md
```

### 启动网页
```bash
# 使用Python启动本地服务器
cd website
python -m http.server 8000

# 或使用Node.js
npx http-server website -p 8000

# 访问地址：http://localhost:8000
```

- **简单易用**: 一键操作，无需复杂设置
