# 📦 蓝河工具箱 - 安装部署指南

## 🚀 快速安装

### 📱 基础安装（推荐）

#### 系统要求
- **Android版本**：7.0 (API 24) 及以上
- **可用存储空间**：50MB
- **RAM内存**：2GB以上

#### 安装步骤
1. **下载APK**
   ```
   从官方网站或应用商店下载最新版APK文件
   版本：蓝河工具箱-v1.5-release.apk
   ```

2. **启用未知来源**
   ```
   设置 > 安全 > 未知来源
   或
   设置 > 应用 > 特殊访问 > 安装未知应用
   ```

3. **安装应用**
   ```
   点击下载的APK文件
   按照提示完成安装
   ```

4. **首次启动**
   ```
   打开应用，授予基础权限
   等待初始化完成
   ```

### ⚡ 高级安装（完整功能）

#### 安装Shizuku框架
Shizuku框架可以提供系统级深度优化功能

**方式一：通过Magisk安装**
```
1. 确保设备已Root
2. 在Magisk Manager中搜索"Shizuku"
3. 下载并安装Shizuku模块
4. 重启设备
5. 启动Shizuku应用并授权
```

**方式二：通过ADB安装**
```
1. 启用USB调试
2. 连接电脑，执行ADB命令：
   adb shell sh /sdcard/Android/data/moe.shizuku.privileged.api/start.sh
3. 启动Shizuku应用
```

#### 配置蓝河工具箱
```
1. 打开蓝河工具箱
2. 点击"Shizuku授权中心"
3. 连接到Shizuku服务
4. 激活高级功能
```

## 🔧 故障排除

### 安装问题

#### APK安装失败
**问题现象**：提示"安装失败"或"解析包时出现问题"

**解决方案**：
1. 检查APK文件完整性
2. 确认未知来源权限已开启
3. 清理应用缓存后重试
4. 尝试使用文件管理器安装

#### 应用闪退
**问题现象**：打开应用后立即关闭

**解决方案**：
1. 强制停止应用后重启
2. 检查系统存储空间
3. 清除应用数据后重启
4. 确认Android版本兼容性

### 功能问题

#### Shizuku连接失败
**问题现象**：提示"Shizuku服务未运行"

**解决方案**：
1. 检查Shizuku应用是否安装
2. 确认Shizuku服务正在运行
3. 重启Shizuku应用
4. 重新授权Shizuku权限

#### 权限获取失败
**问题现象**：功能提示权限不足

**解决方案**：
1. 检查系统权限设置
2. 确认Shizuku权限正常
3. 重启设备后重新尝试
4. 查看系统日志了解详情

#### 优化功能无效
**问题现象**：执行优化后无明显效果

**解决方案**：
1. 检查设备兼容性
2. 确认所有权限已授予
3. 查看优化日志
4. 尝试手动调节相关设置

## 🔄 更新与维护

### 版本更新
```
自动更新：
1. 设置 > 自动更新 > 开启
2. 应用会自动检测新版本
3. 下载完成后提示安装

手动更新：
1. 访问官方网站
2. 下载最新APK
3. 覆盖安装（保留数据）
```

### 数据备份
```
重要数据备份：
1. 设置 > 备份恢复
2. 选择备份内容
3. 执行备份操作
4. 保存备份文件到安全位置
```

### 日志管理
```
日志查看：
1. 设置 > 高级 > 日志记录
2. 选择日志级别
3. 导出日志文件用于分析

日志清理：
1. 设置 > 存储 > 清理缓存
2. 删除旧的日志文件
3. 重置日志记录
```
