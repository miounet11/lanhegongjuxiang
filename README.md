## 🔧 安装与配置

### 基础安装
1. 下载NEURAL APK文件
2. 启用"未知来源"安装权限
3. 安装应用并授予基础权限

### 高级配置（推荐）
1. 安装Shizuku应用并启动服务
2. 在NEURAL中激活Shizuku权限
3. 启用AI学习模式
4. 进行首次系统深度扫描
