# 蓝河工具箱 v1.0 - 安装说明

## 📦 安装包信息
- **文件名**: `蓝河工具箱-v1.0-release.apk`
- **大小**: 7.0MB
- **版本**: 1.0
- **状态**: 已签名发布版本

## 📱 安装方法

### 方法1: 直接安装
1. 将 `蓝河工具箱-v1.0-release.apk` 文件传输到您的Android设备
2. 在文件管理器中找到该APK文件
3. 点击安装
4. 如果出现安全提示，请允许安装未知来源的应用

### 方法2: ADB安装（开发者选项）
```bash
adb install -r 蓝河工具箱-v1.0-release.apk
```

## ✨ 功能特性

### 🎯 核心功能
- **系统性能监控**: 实时监控CPU、内存、电池状态
- **一键智能优化**: 深度清理系统缓存和临时文件
- **核心性能优化**: FPS提升、延迟优化、下载加速、视频防卡
- **网络抓包分析**: 实时监控网络流量和协议分析

### 🔧 系统工具
- **应用管理**: 应用安装、卸载、权限管理
- **电池管理**: 智能省电算法
- **存储管理**: 存储空间分析和清理
- **网络诊断**: WiFi信号分析和网络优化

### 🎨 用户界面
- **现代化设计**: iOS风格的简洁界面
- **深色模式支持**: 自动跟随系统主题
- **响应式布局**: 适配各种屏幕尺寸
- **流畅动画**: Material Design 3动画效果

## 🔐 权限说明

应用需要以下权限来提供完整功能：
- **网络权限**: 用于网络诊断和流量监控
- **存储权限**: 用于应用管理和文件清理
- **系统权限**: 通过Shizuku框架实现高级功能

## 🛠️ 系统要求

- **Android版本**: 7.0 (API 24) 或更高
- **架构支持**: ARM64, ARM32
- **存储空间**: 至少50MB可用空间
